PROJECT PORTFOLIO

A showcase of systems, concepts, and technical documentation illustrating strong creative and logical thinking.

1. CARDINAV: The Cardinal Navigator

Role: Conceptualization, Technical Documentation, Stakeholder Pitching

CARDINAV is a **navigation system** designed to provide **accurate and reliable directional guidance**. It integrates user-friendly interfaces and strategic planning features to help users navigate efficiently, whether in real-world spaces or digital simulations. The core focus was on robust technical documentation and a compelling pitch script to present the system to stakeholders.

2. Applesauce: Modular Productivity Tool

Role: Concept Design, UI/UX Prototyping, Usability Focus

Applesauce is a desktop productivity tool that allows users to manage notes, to-dos, and calendars in a single **modular interface**. My contribution focused on **customization and efficiency**, ensuring the design helps users organize tasks and information quickly, supporting superior workflow management.

3. Automatic Weather Risk Management System

Role: Research, System Design, Documentation

This system monitors and analyzes weather data to identify potential risks, helping users or organizations take **preventive action**. It aims to mitigate the effects of adverse weather events by providing timely alerts and recommendations for **risk management** based on system analysis.

Role: Research, Conceptualization, Technical Documentation

SafeSplit is a secure and user-friendly system for **dividing resources or tasks**. It ensures **safety, transparency, and simplicity** in allocation processes, making it ideal for collaborative environments or resource-sharing scenarios within the academic setting.
